# CGPA Calculator C++
**Author:** Jessen Forbush  
**Technologies:** C++20, Make  
**Methodologies and Design:** OOP, Single Responsibility Principle  

## Program Description
Calculates a given student's GPA (Grade Point Average) based on course credits and grade points. The program allows multiple options for looking up student info, course info, and more. A registry keeps track of all students and courses and provides a clean interactable interface to perform simple calculations.

## How to Build
```bash
make
```

## How to Run
```bash
./bin/main
```
